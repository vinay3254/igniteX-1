Semantic HTML

Structure Accessibility

The Box Model

Every element is a box. Understanding content, padding, border, and margin is fundamental to controlling layout and spacing on any page.

Layout Spacing

Flexbox

The one-dimensional layout tool. Used throughout this page for navbars, card rows, and aligning items — with properties like justify-content, align-items, and flex-wrap.

Flexbox Alignment

CSS Grid

Two-dimensional layouts made easy. The concepts and stats sections on this page are built with Grid — using grid-template-columns and gap for clean, structured designs.

Grid Layout

Using media queries and relative units like rem, %, and vw to ensure the page looks great on phones, tablets, and desktops alike.

Media Queries Mobile First
