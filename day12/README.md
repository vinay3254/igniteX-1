# React Beginner Course Workspace

This is an original React practice project that mirrors the same beginner topic flow as the linked course (without copying protected source code verbatim).

## Run

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

## File structure

```text
react-full-course-original/
├─ public/
├─ src/
│  ├─ App.jsx
│  ├─ App.css
│  ├─ index.css
│  ├─ main.jsx
│  └─ lessons.jsx
├─ index.html
├─ package.json
└─ vite.config.js
```
