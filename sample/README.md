# 🚀 Vinay GK — Portfolio Website

A complete, responsive personal portfolio website built with semantic HTML5, well-organized CSS3, and vanilla JavaScript.

## 📁 Project Structure

```
vinaygk/
├── index.html          # Home page
├── about.html          # About page
├── projects.html       # Projects page
├── contact.html        # Contact page
├── README.md           # Project documentation
├── css/
│   ├── reset.css       # CSS reset / normalize
│   ├── variables.css   # CSS custom properties (design tokens)
│   └── style.css       # Main stylesheet
└── js/
    └── main.js         # Interactivity & animations
```

## ✨ Features

- ✅ Semantic HTML5 structure
- ✅ CSS Custom Properties (variables)
- ✅ Flexbox & CSS Grid layouts
- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Smooth scroll animations (IntersectionObserver)
- ✅ Mobile hamburger navigation
- ✅ Typing effect on hero section
- ✅ Skill bar animations
- ✅ Project filter by category
- ✅ Contact form with validation
- ✅ Scroll-to-top button
- ✅ Accessible (ARIA labels, semantic roles)

## 🚀 Pages

| Page | Description |
|------|-------------|
| `index.html` | Hero, About preview, Skills, Featured Projects, CTA |
| `about.html` | Bio, Education, Certifications, Values |
| `projects.html` | All projects with filter functionality |
| `contact.html` | Contact form, details, FAQ |

## 🎨 Design System

| Token | Value |
|-------|-------|
| Primary Color | `#0f4c81` (Navy Blue) |
| Secondary Color | `#f7941d` (Orange) |
| Accent Color | `#2ecc71` (Green) |
| Font Heading | Poppins |
| Font Body | Inter |

## 🛠️ Technologies Used

- HTML5 (semantic elements: header, nav, main, section, article, footer)
- CSS3 (custom properties, Flexbox, Grid, animations, media queries)
- Vanilla JavaScript (IntersectionObserver, form validation, DOM manipulation)
- Google Fonts (Poppins + Inter)

## 📱 Responsive Breakpoints

| Breakpoint | Target |
|-----------|--------|
| `≤ 1024px` | Tablets – single column layout |
| `≤ 768px` | Mobile – hamburger nav, stacked layout |
| `≤ 480px` | Small phones – full-width buttons |
